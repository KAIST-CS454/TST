# TST Parser

## Abstract
Our topic is "GUI Test Suite Generation Using ACO with EFG Traversal Criterion".

---

# How to Use

## Shell Command
```bash
$ python3 main.py [EFG_file_name].xml
```
## Output
```bash
$ python3 main.py [EFG_file_name].xml
ACO
Random
$
```
After running the code, both **ACO.csv**, **Random.csv** files are generated.
